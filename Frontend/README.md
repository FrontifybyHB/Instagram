# N22 Social Frontend (Vite + React Router)

## Quick start

```bash
# From repo root
cd Frontend
npm install
npm run dev
```

- App runs at http://localhost:5173
- Routes available:
  - /register
  - /login
  - /home
  - /chat
  - /conversation
  - /create-post
  - /profile
  - /user-search

## Build & preview

```bash
npm run build
npm run preview
```
